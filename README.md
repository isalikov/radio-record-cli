## Installation

### Homebrew (macOS/Linux)

```bash

```

### Arch Linux (AUR)

```bash

```

### Debian/Ubuntu


### Fedora/RHEL


### Keybindings

| Key | Action |
|-----|--------|
| `j` / `â†“` | Move down |
| `k` / `â†‘` | Move up |
| `g` | Go to top |
| `G` | Go to bottom |
| `Enter` / `Space` | Play station |
| `s` | Stop playback |
| `+` / `=` | Volume up |
| `-` / `_` | Volume down |
| `/` | Start search |
| `n` | Next search match |
| `N` | Previous search match |
| `Esc` | Clear search |
| `Tab` | Next genre |
| `Shift+Tab` | Previous genre |
| `0` | Reset all filters |
| `f` | Toggle favorite |
| `F` | Show only favorites |
| `1-9` | Play favorite #1-9 |
| `?` | Show help |
| `q` | Quit |

## Configuration

Config is stored at:
- macOS: `~/Library/Application Support/radio-record-cli/config.json`
- Linux: `~/.config/radio-record-cli/config.json`

```json
{
  "favorites": [15016, 15018, 15020],
  "volume": 80
}
```
